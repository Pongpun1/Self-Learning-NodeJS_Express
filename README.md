## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) installed
- npm (comes with Node.js)

### 1. Clone the Repository

```
git clone https://github.com/Pongpun1/Self-Learning-NodeJS_Express.git
cd Self-Learning-NodeJS_Express
```

### 2. Install Dependencies

```
npm install
```

### 3. Run the Application

```
node app.js
```

Or, if `nodemon` is set up:

```
npx nodemon app.js
```

The server should then be running locally (check `app.js` for the configured port).
